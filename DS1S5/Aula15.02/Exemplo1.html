Explicação de alguns elementos em HTML

Elemento p

Tags: <p> e <p/>

O elemento p permite criar parágrafos em uma página HTML.

Elemento br

Tags: <br/>

O elemento br permite inserir uma quebra de linha em uma página HTML.
Observe que a tag <br/> é uma tag vazia (não possui tag de fechamento)
